以下のツールを dotfiles に追加してください。

## ツール情報

$ARGUMENTS

## 手順

ツール名・用途・オプション情報を上記から読み取り、以下の各ステップを順番に実行してください。

---

### 1. Brewfile への追記

`homebrew/Brewfile` を読み込み、適切な位置に追記します。

- `brew "ツール名"` の形式
- 既存のエントリと**重複しないか**確認してから追記
- アルファベット順を崩さないよう近い位置に挿入（厳密でなくてよい）

---

### 2. エイリアスの追加（必要な場合）

`zsh/.zsh/aliases.zsh` を読み込み、エイリアスが有用なら追記します。

- 既存のエイリアスと衝突しないか確認
- 不要（ツール名が短い・エイリアスの慣習がない）なら省略してよい

---

### 3. .zshrc への初期化コードの追加（必要な場合）

`zsh/.zshrc` を読み込み、`eval "$(tool init zsh)"` のような初期化コードが必要なら追記します。

- 既に類似の初期化コードがある場合はスキップ
- 追記位置はファイル末尾の既存の初期化コード群の近く

---

### 4. 補完スクリプトの追加（必要な場合）

ツールが専用の zsh 補完スクリプトを提供している場合、`zsh/.zsh/completions/` への追加方法を案内します。

- 自動補完（`eval` や init スクリプト内）で完結するなら不要
- 手動配置が必要な場合はコマンドを提示する

---

### 5. 完了報告

変更したファイルと内容を要約して報告してください。
その後、以下を案内してください：

```bash
brew install <ツール名>
```

setup.md への反映が必要な変更であれば `/export` の実行も促してください。
